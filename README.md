# Binary Name Converter in C
This project converts a user's name into its binary ASCII representation.

## Example
Input: DJ  
Output: 01000100 01001010

## How to Run
gcc binary.c -o binary  
./binary


# Password Strength Checker 🔐

A simple Python tool that checks the strength of a password and gives suggestions to improve it.

## Features
- Rates password as Weak, Medium, or Strong
- Checks for:
  - Minimum length (8 characters)
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Provides improvement tips

## Usage
```bash
python password_checker.py
